# 🗺️ Tactical Ops — GTA V Military Map

Production-grade tactical operations map for GTA V / FiveM.

## Live Demo
👉 https://provoda111.github.io/military-map

## Features
- 22 marker types (Friendly / Enemy / Tactical)
- 2 map layers: Satellite + Atlas
- Draw lines & zones
- Measure distances
- Import / Export JSON
- Search, filter, layers
- Undo (50 steps)
- Minimap
- Keyboard shortcuts (S/P/L/Z/M/D/F/±/Ctrl+Z/Esc)
